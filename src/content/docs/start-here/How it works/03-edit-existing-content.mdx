Erfahren Sie, wie Sie bestehende Website-Inhalte in Sanity Studio finden, bearbeiten und veröffentlichen.

<br />

<Steps>
1. **Navigieren Sie zu dem Inhalt, den Sie bearbeiten möchten.**
   Klicken Sie beispielsweise in der linken Seitenleiste auf **Seiten**, um auf alle Website-Seiten zuzugreifen.

2. **Wählen Sie den Inhaltstyp aus.**
   Klicken Sie unter **Seiten** auf **Seite**, um die Liste der verfügbaren Seiten zu sehen.

3. **Wählen Sie die spezifische Seite aus, die Sie bearbeiten möchten.**
   Der Seiten-Editor wird auf der rechten Seite des Bildschirms geöffnet.

4. **Wählen Sie die Komponente aus, die Sie ändern möchten.**
   Scrollen Sie durch den Seiteninhalt und klicken Sie auf die Komponente, die Sie ändern möchten.

   <figure style="text-align: center;">
     <img
       src={sanityStudioWorkflowExample.src}
       alt="Sanity Studio Beispiel für das Bearbeiten bestehender Inhalte"
     />
     <figcaption>
       <em>Sanity Studio Beispiel für das Bearbeiten bestehender Inhalte</em>
     </figcaption>
   </figure>

   <br />

5. **Bearbeiten Sie den Komponenteninhalt.**
   Der Komponenten-Editor wird geöffnet. Klicken Sie in ein beliebiges Textfeld, Bild oder eine Einstellung, um Ihre Änderungen vorzunehmen. Ihre Bearbeitungen werden automatisch als Entwurf gespeichert.

   <figure style="text-align: center;">
     <img
       src={sanityStudioEditComponent.src}
       alt="Sanity Studio Beispiel für das Bearbeiten von Komponenteninhalten"
     />
     <figcaption>
       <em>
         Sanity Studio Beispiel für das Bearbeiten von Komponenteninhalten
       </em>
     </figcaption>
   </figure>

   <br />

6. **Bearbeiten Sie Elemente innerhalb einer Komponente (falls zutreffend).**
   Einige Komponenten enthalten mehrere Elemente (wie Teammitglieder in einem Team-Abschnitt oder Bilder in einer Galerie). Klicken Sie auf ein einzelnes Element, um es zu bearbeiten.

   <figure style="text-align: center;">
     <img
       src={sanityStudioEditComponentItem.src}
       alt="Sanity Studio Beispiel für das Bearbeiten eines bestehenden Komponentenelements"
     />
     <figcaption>
       <em>
         Sanity Studio Beispiel für das Bearbeiten eines bestehenden
         Komponentenelements
       </em>
     </figcaption>
   </figure>

   <br />

7. **Veröffentlichen Sie Ihre Änderungen.**
   Sobald Sie mit Ihren Bearbeitungen zufrieden sind, klicken Sie auf die Schaltfläche **Veröffentlichen** am unteren Rand des Editors, um Ihre Änderungen auf der Live-Website sichtbar zu machen.

</Steps>
