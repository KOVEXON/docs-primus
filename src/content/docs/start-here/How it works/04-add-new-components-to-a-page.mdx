Sie können ganze Abschnitte (wie Hero-Abschnitte, Galerien oder Testimonials) zu jeder Seite hinzufügen.

<br />

<Steps>
1. **Öffnen Sie die Seite**, auf der Sie eine Komponente hinzufügen möchten, indem Sie durch die linke Seitenleiste navigieren.

2. **Scrollen Sie nach unten** zum unteren Bereich des Seiteninhalts im Editor.

3. **Klicken Sie auf die Schaltfläche "Element hinzufügen"** (normalerweise mit einem + Symbol gekennzeichnet).

4. **Wählen Sie den Komponententyp aus**, den Sie hinzufügen möchten, aus der Liste (zum Beispiel: Hero-Abschnitt, Galerie, Testimonials oder Textblock).

5. **Fügen Sie Inhalte zu Ihrer neuen Komponente hinzu** (siehe Schritt 6 unten).

6. **Ordnen Sie bei Bedarf neu an.**
   Die neue Komponente erscheint am Ende der Seite. Sie können sie an eine andere Position ziehen, indem Sie auf den Verschiebegriff klicken und ihn gedrückt halten (normalerweise auf der linken Seite der Komponente).

7. **Veröffentlichen Sie die Seite**, um die neue Komponente auf Ihrer Website sichtbar zu machen.

</Steps>
