Erfahren Sie Schritt für Schritt, wie Sie in Sanity Studio eine neue Seite anlegen, ausfüllen und veröffentlichen.

<br />

<Aside type="tip">
  Erstellen Sie zuerst die Seite und veröffentlichen Sie sie anschließend.
  Inhalte und Komponenten können Sie danach jederzeit ergänzen oder anpassen.
</Aside>

<br />

<Steps>
1. **Öffnen Sie in der linken Seitenleiste den Bereich „Seiten“.**
   Dort finden Sie alle bestehenden Seiten Ihrer Website.

2. **Wähle den Dokumenttyp „Seite“ oder „Rechtliche Seite“ aus.**

3. **Klicken Sie oben rechts auf das „+“-Symbol (Neues Dokument erstellen)**.

4. **Geben Sie den Seitentitel ein.**
   Verwenden Sie einen klaren Namen (z. B. „Über uns“ oder „Leistungen“), damit Sie die Seite später leicht wiederfinden.

5. **Setzen Sie den Slug (URL-Pfad).**
   Der Slug bestimmt die URL der Seite.
   Klicke auf „Generate“, um automatisch eine URL zu erstellen (z. B. /ueber-uns).
   Du kannst den Slug bei Bedarf anpassen.

6. **Füllen Sie die wichtigsten Seitenfelder aus.**
   Je nach Projekt gehören dazu z. B. Seitentitel, Kurzbeschreibung, Header-Einstellungen oder erste Inhaltsabschnitte.

7. **Fügen Sie die ersten Komponenten hinzu.**
   Scrollen Sie zum Inhaltsbereich und klicken Sie auf **Element hinzufügen**, um z. B. Hero, Textblock oder Galerie einzufügen.

8. **Überprüfen Sie Ihre Inhalte im Editor.**
   Kontrollieren Sie Texte, Bilder, Reihenfolge der Komponenten und Links.

9. **Klicken Sie auf „Veröffentlichen“.**
   Erst danach ist die neue Seite auf Ihrer Live-Website sichtbar.

</Steps>

<Aside type="tip">
  - Verwende klare und kurze Titel.
  - Achte auf verständliche URLs (Slug).
  - Halte Inhalte übersichtlich und strukturiert.
  - Prüfe die Seite immer vor dem Veröffentlichen.
</Aside>
