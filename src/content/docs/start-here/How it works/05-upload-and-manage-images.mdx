Erfahren Sie, wie Sie Bilder in Sanity Studio hochladen, ersetzen und verwalten, einschließlich des Hinzufügens von Alt-Text für Barrierefreiheit und SEO.

<br />

<Steps>
1. **Klicken Sie auf das Bildfeld**, das Sie bearbeiten möchten (zeigt normalerweise einen Platzhalter oder ein vorhandenes Bild an).

2. **Laden Sie ein Bild hoch oder wählen Sie eines aus:**

   - Klicken Sie auf **Hochladen**, um ein neues Bild von Ihrem Computer hinzuzufügen
   - Oder klicken Sie auf **Auswählen**, um ein vorhandenes Bild aus der Medienbibliothek auszuwählen

3. **Fügen Sie Alt-Text hinzu (empfohlen).**
   Wenn ein Alt-Text-Feld verfügbar ist, fügen Sie eine kurze Beschreibung des Bildes hinzu. Dies hilft Menschen, die Screenreader verwenden, und verbessert die Sichtbarkeit in Suchmaschinen (SEO).

   {" "}

   <Aside type="tip">
     Gutes Alt-Text-Beispiel: "Team-Meeting in hellem Büroraum" anstatt nur
     "Büro"
   </Aside>

4. **Veröffentlichen Sie Ihre Änderungen**, um das neue Bild auf der Website sichtbar zu machen.

</Steps>
