---
title: Was bedeutet CMS?
description: Erfahren Sie, was CMS bedeutet
---

CMS steht für **Content Management System** (Inhaltsverwaltungssystem).

Einfach ausgedrückt ist ein CMS ein Tool, mit dem Sie **Website-Inhalte einfach verwalten** können, ähnlich wie beim Ausfüllen eines Formulars.

Anstatt komplizierten Code zu bearbeiten:

- Geben Sie Text in Felder ein
- Laden Sie Bilder hoch
- Klicken Sie auf Speichern

Sanity Studio ist das **CMS, das für Ihre Website verwendet wird**.
