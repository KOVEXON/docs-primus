Die **Pricing Section** ist ein Content-Block zur Präsentation von Preisen, Paketen oder Angeboten in übersichtlichen Preis-Karten.

## Felder

### Headline

* **Typ**: String
* **Pflichtfeld**: Nein
* **Beschreibung**: Hauptüberschrift für die Pricing-Sektion

### Description

* **Typ**: Text
* **Pflichtfeld**: Nein
* **Zeilen**: 4
* **Beschreibung**: Beschreibung für die Pricing-Sektion

### Items

* **Typ**: Array von Pricing-Item-Objekten
* **Pflichtfeld**: Nein
* **Beschreibung**: Liste der Preis-Pakete

#### Pricing-Item Felder

##### Item Name

* **Typ**: String
* **Pflichtfeld**: Ja
* **Beschreibung**: Name des Preis-Pakets

##### Price

* **Typ**: String
* **Pflichtfeld**: Nein
* **Beschreibung**: Preis (z. B. "€1.500" oder "Auf Anfrage")

##### Description

* **Typ**: Text
* **Pflichtfeld**: Nein
* **Zeilen**: 4
* **Beschreibung**: Beschreibung des Pakets

##### Features

* **Typ**: Array von Strings
* **Pflichtfeld**: Nein
* **Beschreibung**: Liste der Features/Inhalte des Pakets

##### Show Button

* **Typ**: Boolean
* **Initialwert**: `false`
* **Beschreibung**: Schalter zum Anzeigen eines Buttons auf dieser Preis-Karte

##### Button

* **Typ**: Object
* **Versteckt**: Ja (wenn Show Button nicht aktiviert ist)
* **Felder**:
  * **Button Text** (String) - Text, der auf dem Button angezeigt wird
  * **Relative Link** (String) - Relativer Pfad (z. B. `/kontakt`, `/anmeldung`)
    * **Validierung**: Muss mit `/` beginnen
  * **Button Role** (String) - Rolle des Buttons (Primary/Secondary)
    * **Pflichtfeld**: Ja
    * **Initialwert**: `primary`

##### Highlighted

* **Typ**: Boolean
* **Pflichtfeld**: Nein
* **Beschreibung**: Markiert dieses Paket als hervorgehoben (z. B. "Empfohlen")

##### Show Badge

* **Typ**: Boolean
* **Initialwert**: `false`
* **Beschreibung**: Schalter zum Anzeigen eines Badges auf dieser Preis-Karte

##### Badge

* **Typ**: Reference zu `badge`
* **Versteckt**: Ja (wenn Show Badge nicht aktiviert ist)
* **Beschreibung**: Badge-Komponente (z. B. "Bestseller", "NEU")

### Background Color

* **Typ**: Reference zu `backgroundColor`
* **Pflichtfeld**: Nein
* **Beschreibung**: Hintergrundfarbe für die gesamte Pricing-Sektion

## Verwendung

Die Pricing Section wird typischerweise verwendet für:

* **Preislisten** - Präsentation von Preisen für verschiedene Angebote
* **Pakete** - Verschiedene Pakete mit unterschiedlichen Features
* **Angebote** - Sonderangebote oder Rabatte
* **Vergleiche** - Vergleich verschiedener Preisoptionen

## Beispiel

Eine typische Pricing Section könnte folgende Struktur haben:

### Headline

"Unsere Preise"

### Description

"Wählen Sie das passende Paket für Ihre Bedürfnisse"

### Items

**Paket 1:**

* **Item Name**: "Basis-Paket"
* **Price**: "€1.200"
* **Description**: "Grundausbildung für den Führerschein Klasse B"
* **Features**:
  * "12 Theoriestunden"
  * "20 Fahrstunden"
  * "Prüfungsvorbereitung"
* **Show Button**: `true`
* **Button**: "Jetzt buchen" → `/anmeldung`
* **Highlighted**: `false`

**Paket 2:**

* **Item Name**: "Premium-Paket"
* **Price**: "€1.800"
* **Description**: "Umfassende Ausbildung mit zusätzlichen Leistungen"
* **Features**:
  * "Alle Basis-Features"
  * "Zusätzliche 10 Fahrstunden"
  * "Intensivkurs-Option"
  * "Persönlicher Betreuer"
* **Show Button**: `true`
* **Button**: "Jetzt buchen" → `/anmeldung`
* **Highlighted**: `true`
* **Show Badge**: `true`
* **Badge**: "Bestseller"

## Best Practices

* **Klarheit**: Klare, verständliche Preisangaben
* **Vergleichbarkeit**: Ähnliche Struktur für alle Pakete
* **Hervorhebung**: Ein Paket als "Empfohlen" hervorheben
* **Features**: Klare Auflistung der enthaltenen Features
* **Buttons**: CTA-Buttons für bessere Conversion
* **Anzahl**: 2-4 Pakete sind optimal
