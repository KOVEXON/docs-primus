Die **Home Hero Section** ist ein spezieller Hero-Bereich, der für die Startseite konzipiert ist. Sie bietet eine flexible Headline-Struktur mit hervorgehobenen Wörtern, farbigen Keywords und Call-to-Action-Buttons.

## Felder

### Headline

* **Typ**: Object
* **Pflichtfeld**: Ja
* **Beschreibung**: Komplexe Headline-Struktur mit mehreren Komponenten

#### Main Part

* **Typ**: String
* **Pflichtfeld**: Ja
* **Beschreibung**: Hauptteil der Headline

#### Highlighted Word

* **Typ**: String
* **Pflichtfeld**: Nein
* **Beschreibung**: Hervorgehobenes Wort in der Headline. Leer lassen, wenn kein Wort hervorgehoben werden soll.

#### Keywords

* **Typ**: Array von Keyword-Objekten
* **Pflichtfeld**: Nein
* **Beschreibung**: Keywords mit individuellen Farben (z. B. "elektrisch", "digital", "modern")

##### Keyword-Objekt Felder

* **Keyword Text** (String, Pflichtfeld) - Der Text des Keywords
* **Color** (String) - Farbe des Keywords
  * Verfügbare Farben: Green (Standard), Orange, Purple, Blue, Red, Pink, Yellow, Gray, Black

### Subheadline

* **Typ**: Text
* **Pflichtfeld**: Nein
* **Beschreibung**: Unterüberschrift unter der Hauptheadline

### Background Image

* **Typ**: Image
* **Pflichtfeld**: Nein
* **Optionen**: Hotspot aktiviert
* **Beschreibung**: Hintergrundbild für den Hero-Bereich

### Call to Action Buttons

* **Typ**: Array von Button-Objekten
* **Pflichtfeld**: Nein
* **Beschreibung**: Call-to-Action-Buttons für den Hero-Bereich
* **Validierung**:
  * Maximal ein Primary-Button erlaubt
  * Maximal ein Secondary-Button erlaubt

#### Button-Objekt Felder

* **Button Text** (String) - Text, der auf dem Button angezeigt wird
* **Relative Link** (String) - Relativer Pfad (z. B. `/speedkurse` oder `/kontakt`)
  * **Validierung**: Muss mit `/` beginnen
* **Button Role** (String) - Rolle des Buttons
  * **Primary** - Haupt-Button (hervorgehoben)
  * **Secondary** - Sekundär-Button

## Verwendung

Die Home Hero Section wird typischerweise verwendet für:

* **Startseite** - Als Haupt-Hero-Bereich auf der Homepage
* **Landing Pages** - Als auffälliger Einstiegsbereich
* **Marketing-Kampagnen** - Mit farbigen Keywords und klaren Call-to-Actions

## Beispiel

Eine typische Home Hero Section könnte folgende Struktur haben:

### Headline

* **Main Part**: "Willkommen bei der modernen Fahrschule"
* **Highlighted Word**: "modernen"
* **Keywords**:
  * "elektrisch" (Farbe: Green)
  * "digital" (Farbe: Blue)
  * "nachhaltig" (Farbe: Green)

### Subheadline

"Wir bereiten Sie optimal auf Ihre Führerscheinprüfung vor - mit modernster Technik und erfahrenen Fahrlehrern."

### Background Image

Ein professionelles Bild einer modernen Fahrschule oder eines Fahrzeugs.

### Call to Action Buttons

1. **Primary Button**: "Jetzt anmelden" → `/anmeldung`
2. **Secondary Button**: "Mehr erfahren" → `/ueber-uns`

## Besonderheiten

* **Komplexe Headline**: Ermöglicht flexible Gestaltung mit hervorgehobenen Wörtern und farbigen Keywords
* **Farbige Keywords**: Jedes Keyword kann individuell eingefärbt werden für visuelle Hervorhebung
* **Button-Validierung**: System stellt sicher, dass nur ein Primary- und ein Secondary-Button verwendet werden
* **Hotspot-Bild**: Hintergrundbild unterstützt Hotspot-Funktion für bessere Bildbearbeitung

## Best Practices

* **Headline**: Kurz und prägnant halten, Highlighted Word sparsam verwenden
* **Keywords**: 2-4 Keywords verwenden, Farben konsistent wählen
* **Subheadline**: Ergänzende Informationen, die die Headline unterstützen
* **Buttons**: Klare, handlungsorientierte Texte verwenden
* **Background Image**: Hochwertiges Bild verwenden, das die Botschaft unterstützt
