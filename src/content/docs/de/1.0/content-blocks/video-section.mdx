Die **Video Section** ist ein Content-Block für die Einbettung von Videos mit optionaler Headline, Beschreibung und Poster-Bild.

## Felder

### Headline

* **Typ**: String
* **Pflichtfeld**: Ja
* **Maximale Länge**: 120 Zeichen
* **Beschreibung**: Hauptüberschrift für die Video-Sektion

### Description

* **Typ**: Text
* **Pflichtfeld**: Nein
* **Zeilen**: 3
* **Beschreibung**: Beschreibungstext für das Video

### Video File

* **Typ**: File
* **Pflichtfeld**: Nein
* **Akzeptierte Formate**: Video/\* (MP4, WebM, etc.)
* **Beschreibung**: Hochgeladenes Video

### Video URL (Alternative)

* **Typ**: URL
* **Pflichtfeld**: Nein
* **Beschreibung**: Alternative: Direkte Video-URL statt Datei-Upload

### Poster Image

* **Typ**: Image
* **Pflichtfeld**: Nein
* **Optionen**: Hotspot aktiviert
* **Beschreibung**: Thumbnail-Bild, das vor dem Abspielen des Videos angezeigt wird

### Background Color

* **Typ**: Reference zu `backgroundColor`
* **Pflichtfeld**: Nein
* **Beschreibung**: Hintergrundfarbe für die Video-Sektion

## Verwendung

Die Video Section wird typischerweise verwendet für:

* **Erklärvideos** - Erklärungen von Dienstleistungen oder Produkten
* **Testimonials** - Video-Bewertungen von Kunden
* **Tutorials** - Anleitungsvideos
* **Präsentationen** - Unternehmenspräsentationen
* **Event-Highlights** - Videos von Veranstaltungen

## Beispiel

Eine typische Video Section könnte folgende Struktur haben:

* **Headline**: "So läuft eine Fahrstunde ab"
* **Description**: "Sehen Sie, wie eine typische Fahrstunde bei uns aussieht"
* **Video File**: Hochgeladenes MP4-Video oder
* **Video URL**: Direkte URL zu einem Video-Hosting-Service
* **Poster Image**: Thumbnail-Bild mit Fahrschule oder Fahrzeug

## Besonderheiten

* **Zwei Video-Quellen**: Video kann hochgeladen oder per URL eingebunden werden
* **Poster-Bild**: Thumbnail-Bild für bessere visuelle Präsentation
* **Hotspot-Bild**: Poster-Bild unterstützt Hotspot-Funktion

## Best Practices

* **Video-Format**: MP4 wird für beste Kompatibilität empfohlen
* **Poster-Bild**: Verwenden Sie ein aussagekräftiges Thumbnail
* **Video-Größe**: Optimieren Sie Videos für Web (komprimiert, aber gute Qualität)
* **Beschreibung**: Geben Sie Kontext zum Video-Inhalt
