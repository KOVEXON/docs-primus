Die **News Section** ist ein Content-Block zur Anzeige von News-Artikeln mit automatischem Abruf und Load-More-Funktionalität.

## Felder

### Section Headline

* **Typ**: String
* **Pflichtfeld**: Nein
* **Beschreibung**: Hauptüberschrift für die News-Sektion

### Section Description

* **Typ**: Text
* **Pflichtfeld**: Nein
* **Beschreibung**: Beschreibung für die News-Sektion

### Maximum Articles to Display

* **Typ**: Number
* **Initialwert**: 6
* **Minimale Wert**: 1
* **Maximale Wert**: 50
* **Beschreibung**: Maximale Anzahl der anzuzeigenden Artikel (Standard: 6)

### Show Load More Button

* **Typ**: Boolean
* **Initialwert**: `true`
* **Beschreibung**: Ob ein "Load More"-Button für zusätzliche Artikel angezeigt werden soll

### Background Color

* **Typ**: Reference zu `backgroundColor`
* **Pflichtfeld**: Nein
* **Beschreibung**: Hintergrundfarbe für die gesamte News-Sektion

## Verwendung

Die News Section wird typischerweise verwendet für:

* **News-Bereiche** - Anzeige von News-Artikeln
* **Aktuelles** - Aktuelle Nachrichten und Updates
* **Blog-Vorschau** - Vorschau von Blog-Artikeln
* **Ankündigungen** - Wichtige Ankündigungen

## Beispiel

Eine typische News Section könnte folgende Konfiguration haben:

* **Section Headline**: "Aktuelles"
* **Section Description**: "Neueste Nachrichten und Updates"
* **Maximum Articles to Display**: 6
* **Show Load More Button**: `true`

## Besonderheiten

* **Automatischer Abruf**: Artikel werden automatisch abgerufen
* **Load More**: Zusätzliche Artikel können nachgeladen werden
* **Sortierung**: Artikel werden nach Veröffentlichungsdatum sortiert

## Best Practices

* **Anzahl**: 6-12 Artikel initial anzeigen
* **Load More**: Aktivieren für bessere Performance bei vielen Artikeln
* **Aktualität**: Regelmäßig neue Artikel veröffentlichen
