Viele Komponenten (wie Team-, Galerie- oder Listen-Abschnitte) können mehrere Elemente enthalten. Hier erfahren Sie, wie Sie Inhaltselemente innerhalb von Komponenten in Sanity Studio hinzufügen und veröffentlichen.

<br />

<Steps>
  1. **Klicken Sie auf die Komponente**, die Sie bearbeiten möchten.
     Dadurch wird der Komponenten-Editor geöffnet.

  2. **Finden Sie den Inhaltsbereich** innerhalb der Komponente (normalerweise beschriftet als "Elemente", "Mitglieder", "Bilder" oder ähnlich).

  3. **Klicken Sie auf "Element hinzufügen"** am Ende der Liste.

  4. **Füllen Sie die erforderlichen Felder aus**, wie Text, Bilder, Links oder andere Einstellungen.

  5. **Speichern und veröffentlichen.**
     Das neue Element wird automatisch als Entwurf gespeichert. Klicken Sie auf **Veröffentlichen**, um es auf Ihrer Website sichtbar zu machen.
</Steps>
