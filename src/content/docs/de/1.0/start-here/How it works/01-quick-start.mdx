Diese Anleitung zeigt Ihnen, wie Sie die Inhalte Ihrer Website mit Sanity Studio verwalten und aktualisieren können. Technische Kenntnisse sind nicht erforderlich.

<Aside>
  **Wichtig:** Alle Änderungen werden automatisch als Entwürfe in Sanity Studio
  gespeichert, aber sie werden erst auf Ihrer Live-Website sichtbar, nachdem Sie
  auf **Veröffentlichen** geklickt haben.
</Aside>

<br />

<br />

<Steps>
  1. ## Öffnen Sie Sanity Studio in Ihrem Browser

     * Öffnen Sie den Link zu Ihrem Sanity Studio in Ihrem Browser `www.example.com/admin`
     * Melden Sie sich mit den Ihnen zur Verfügung gestellten Zugangsdaten an

     <figure style="text-align: center;">
       <img src={sanityStudioUserLogin.src} alt="Sanity Studio Anmeldeseite" />

       <figcaption>
         <em>Sanity Studio Anmeldeseite</em>
       </figcaption>
     </figure>

     <br />

     Nach der Anmeldung sehen Sie das Sanity Studio Dashboard mit allen verfügbaren Inhaltstypen.

  2. ## Navigieren Sie im Dashboard

     In der linken Seitenleiste finden Sie verschiedene Inhaltsbereiche, wie zum Beispiel:

     * Top Header
     * Header
     * Footer
     * Firmeninformationen
     * Kartenbereich
     * Veranstaltungsliste
     * Urlaubsbenachrichtigung
     * Werbebanner
     * Blog
     * Download-Bereich
     * Seiten
     * Website-Einstellungen

     <figure style="text-align: center;">
       <img src={sanityStudioDashboard.src} alt="Sanity Studio Dashboard" />

       <figcaption><em>Sanity Studio Dashboard</em></figcaption>
     </figure>

     <br />

     Klicken Sie auf einen beliebigen Bereich, um dessen Inhalte anzuzeigen und zu bearbeiten.
</Steps>
