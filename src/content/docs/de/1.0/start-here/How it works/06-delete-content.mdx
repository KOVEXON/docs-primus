Wenn Sie eine Seite, Komponente oder ein Element nicht mehr benötigen, können Sie es dauerhaft entfernen.

<br />

<Steps>
  1. **Öffnen Sie das Inhaltselement**, das Sie löschen möchten (navigieren Sie über die linke Seitenleiste dorthin).

  2. **Klicken Sie auf die Schaltfläche "Löschen"** (normalerweise am unteren Rand des Editors oder in einem Menü mit drei Punkten zu finden).

  3. **Veröffentlichen Sie die Änderungen**, um den Inhalt von Ihrer Live-Website zu entfernen.
</Steps>

<br />

<Aside type="danger">
  **Warnung:** Das Löschen von Inhalten ist dauerhaft und kann nicht rückgängig
  gemacht werden. Wenn Sie sich unsicher sind, sollten Sie erwägen, den Inhalt
  zu depublizieren oder zu bearbeiten, anstatt ihn zu löschen.
</Aside>
