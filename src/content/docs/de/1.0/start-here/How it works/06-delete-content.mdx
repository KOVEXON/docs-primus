If you no longer need a page, component, or item, you can remove it permanently.

<br />

<Steps>
  1. **Open the content item** you want to delete (navigate to it using the left sidebar).

  2. **Click the "Delete" button** (usually found at the bottom of the editor or in a menu with three dots).

  3. **Publish the changes** to remove the content from your live website.
</Steps>
