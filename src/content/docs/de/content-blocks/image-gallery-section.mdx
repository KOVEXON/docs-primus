Die **Image Gallery Section** ist ein Content-Block für die Präsentation von Bildergalerien mit Titel, Untertitel und mehreren Bildern.

## Felder

### Title

- **Typ**: String
- **Pflichtfeld**: Ja
- **Beschreibung**: Titel der Galerie

### Subtitle

- **Typ**: String
- **Pflichtfeld**: Nein
- **Beschreibung**: Untertitel der Galerie

### Images

- **Typ**: Array von Bildern
- **Pflichtfeld**: Nein
- **Minimale Anzahl**: 1
- **Maximale Anzahl**: 20
- **Beschreibung**: Liste der Galeriebilder

#### Bild-Felder

- **Alt Text** (String) - Wichtig für Barrierefreiheit und SEO
- **Caption** (String) - Optionale Bildunterschrift

### Background Color

- **Typ**: Reference zu `backgroundColor`
- **Pflichtfeld**: Nein
- **Beschreibung**: Hintergrundfarbe für die Galerie-Sektion

## Verwendung

Die Image Gallery Section wird typischerweise verwendet für:

- **Produktgalerien** - Präsentation von Produkten
- **Event-Galerien** - Bilder von Veranstaltungen
- **Portfolio** - Portfolio-Präsentationen
- **Referenzen** - Referenzbilder

## Beispiel

Eine typische Image Gallery Section könnte folgende Struktur haben:

- **Title**: "Unsere Fahrzeuge"
- **Subtitle**: "Moderne Fahrzeuge für Ihre Ausbildung"
- **Images**: Array von Fahrzeugbildern mit Alt-Text und optionalen Captions

## Best Practices

- **Alt-Text**: Immer aussagekräftigen Alt-Text für jedes Bild ausfüllen
- **Bildqualität**: Hochwertige, konsistente Bilder verwenden
- **Anzahl**: 6-12 Bilder sind optimal für die Benutzererfahrung
- **Captions**: Verwenden Sie Captions für zusätzlichen Kontext
