Die **Testimonials Section** ist ein Content-Block zur Anzeige von Kundenbewertungen, Referenzen und Testimonials.

## Felder

### Title

- **Typ**: String
- **Pflichtfeld**: Nein
- **Beschreibung**: Titel für die Testimonials-Sektion

### Headline

- **Typ**: String
- **Pflichtfeld**: Ja
- **Beschreibung**: Hauptüberschrift für die Testimonials-Sektion

### Testimonials

- **Typ**: Array von Testimonial-Objekten
- **Pflichtfeld**: Nein
- **Beschreibung**: Liste der Testimonials

#### Testimonial-Objekt Felder

##### Quote

- **Typ**: Text
- **Pflichtfeld**: Nein
- **Beschreibung**: Zitat oder Bewertungstext

##### Author

- **Typ**: String
- **Pflichtfeld**: Nein
- **Beschreibung**: Name des Autors der Bewertung

##### Show Avatar

- **Typ**: Boolean
- **Initialwert**: `true`
- **Beschreibung**: Schalter zum Anzeigen/Verstecken des Avatars

##### Avatar

- **Typ**: Image
- **Versteckt**: Ja (wenn Show Avatar nicht aktiviert ist)
- **Beschreibung**: Profilbild des Autors

### Background Color

- **Typ**: Reference zu `backgroundColor`
- **Pflichtfeld**: Nein
- **Beschreibung**: Hintergrundfarbe für die gesamte Testimonials-Sektion

## Verwendung

Die Testimonials Section wird typischerweise verwendet für:

- **Kundenbewertungen** - Anzeige von Kundenbewertungen
- **Referenzen** - Präsentation von Referenzen
- **Erfolgsgeschichten** - Erfolgsgeschichten von Kunden
- **Vertrauensbildung** - Aufbau von Vertrauen durch externe Bestätigung

## Beispiel

Eine typische Testimonials Section könnte folgende Struktur haben:

### Title

"Was unsere Kunden sagen"

### Headline

"Kundenbewertungen"

### Testimonials

**Testimonial 1:**

- **Quote**: "Die beste Fahrschule in der Region! Sehr professionell und geduldig."
- **Author**: "Max Mustermann"
- **Show Avatar**: `true`
- **Avatar**: Profilbild

**Testimonial 2:**

- **Quote**: "Schnell und erfolgreich zum Führerschein. Kann ich nur empfehlen!"
- **Author**: "Anna Schmidt"
- **Show Avatar**: `true`
- **Avatar**: Profilbild

## Best Practices

- **Authentizität**: Verwenden Sie echte, authentische Bewertungen
- **Vielfalt**: Zeigen Sie verschiedene Perspektiven und Erfahrungen
- **Avatare**: Profilbilder erhöhen die Glaubwürdigkeit
- **Anzahl**: 3-6 Testimonials sind optimal
- **Aktualität**: Regelmäßig aktualisieren mit neuen Bewertungen
