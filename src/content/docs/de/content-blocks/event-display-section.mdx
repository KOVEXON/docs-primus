---
title: Event Display Section
description: Content-Block zur Anzeige von Veranstaltungen in Tabellenform
---

Die **Event Display Section** ist ein Content-Block zur Anzeige von Veranstaltungen aus einer Events List in übersichtlicher Tabellenform.

## Felder

### Section Headline

- **Typ**: String
- **Pflichtfeld**: Nein
- **Beschreibung**: Hauptüberschrift für die Events-Sektion

### Section Description

- **Typ**: Text
- **Pflichtfeld**: Nein
- **Zeilen**: 3
- **Beschreibung**: Optionale Beschreibung unter der Headline

### Date Column Title

- **Typ**: String
- **Initialwert**: "Tag & Datum"
- **Beschreibung**: Titel für die Datumsspalte

### Title Column Title

- **Typ**: String
- **Initialwert**: "Unterricht"
- **Beschreibung**: Titel für die Session-Titel-Spalte

### Time Column Title

- **Typ**: String
- **Initialwert**: "Uhrzeit"
- **Beschreibung**: Titel für die Zeit-Spalte

### Available Status Text

- **Typ**: String
- **Initialwert**: "Plätze verfügbar"
- **Beschreibung**: Text, der angezeigt wird, wenn Plätze verfügbar sind

### Full Status Text

- **Typ**: String
- **Initialwert**: "Ausgebucht"
- **Beschreibung**: Text, der angezeigt wird, wenn die Veranstaltung voll belegt ist

### Events List

- **Typ**: Reference zu `eventsList`
- **Pflichtfeld**: Ja
- **Beschreibung**: Referenz zu einem Events List-Dokument

### No Events Message

- **Typ**: String
- **Pflichtfeld**: Nein
- **Beschreibung**: Nachricht, die angezeigt wird, wenn keine Veranstaltungen vorhanden sind

### Background Color

- **Typ**: Reference zu `backgroundColor`
- **Pflichtfeld**: Nein
- **Beschreibung**: Hintergrundfarbe für die gesamte Events-Sektion

## Verwendung

Die Event Display Section wird typischerweise verwendet für:

- **Veranstaltungskalender** - Anzeige von Veranstaltungen und Terminen
- **Kursübersicht** - Übersicht von Kursen und Terminen
- **Event-Seiten** - Detaillierte Event-Listen

## Beispiel

Eine typische Event Display Section könnte folgende Konfiguration haben:

- **Section Headline**: "Aktuelle Veranstaltungen"
- **Section Description**: "Hier finden Sie alle aktuellen Kurse und Termine"
- **Date Column Title**: "Datum"
- **Title Column Title**: "Kurs"
- **Time Column Title**: "Uhrzeit"
- **Available Status Text**: "Plätze verfügbar"
- **Full Status Text**: "Ausgebucht"
- **Events List**: Referenz zu einem Events List-Dokument
- **No Events Message**: "Derzeit sind keine Veranstaltungen geplant"

## Besonderheiten

- **Tabellenform**: Veranstaltungen werden in übersichtlicher Tabellenform angezeigt
- **Status-Anzeige**: Zeigt Verfügbarkeit der Plätze an
- **Referenzierung**: Verwendet Events List-Dokumente für zentrale Verwaltung
- **Anpassbare Spaltentitel**: Spaltentitel können angepasst werden

## Best Practices

- **Klarheit**: Klare, verständliche Spaltentitel
- **Status-Texte**: Verständliche Status-Texte für Verfügbarkeit
- **Aktualität**: Regelmäßig Events List aktualisieren
- **Nachricht**: Freundliche Nachricht, wenn keine Events vorhanden sind
