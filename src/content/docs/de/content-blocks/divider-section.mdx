---
title: Divider Section
description: Trennlinie oder Abstand zwischen Content-Blöcken
---

Die **Divider Section** ist ein einfacher Content-Block, der eine Trennlinie oder Abstand zwischen anderen Content-Blöcken erzeugt.

## Felder

### Show Divider Line

- **Typ**: Boolean
- **Pflichtfeld**: Nein
- **Beschreibung**: Schalter zum Anzeigen/Verstecken der Trennlinie

### Divider Width

- **Typ**: String
- **Initialwert**: `100%`
- **Optionen**:
  - **100%** - Volle Breite
  - **Container Width (max-w-6xl)** - Container-Breite
  - **Custom Width** - Benutzerdefinierte Breite
- **Beschreibung**: Steuert die Breite der Trennlinie

### Custom Width

- **Typ**: String
- **Pflichtfeld**: Nein
- **Versteckt**: Ja (wenn Divider Width nicht auf "Custom Width" gesetzt ist)
- **Beschreibung**: Benutzerdefinierter CSS-Breitenwert (z. B. "400px", "50vw", "20rem")
- **Beschreibung**: Wird nur verwendet, wenn Breite auf "Custom Width" gesetzt ist

## Verwendung

Die Divider Section wird typischerweise verwendet für:

- **Visuelle Trennung** - Trennung zwischen verschiedenen Content-Bereichen
- **Abstand** - Erzeugung von Abstand zwischen Blöcken
- **Strukturierung** - Visuelle Strukturierung von Seiteninhalten

## Beispiel

Eine typische Divider Section könnte folgende Konfiguration haben:

- **Show Divider Line**: `true`
- **Divider Width**: `100%`

Oder für einen Abstand ohne Linie:

- **Show Divider Line**: `false`
- **Divider Width**: `100%`

## Best Practices

- **Sparsamkeit**: Verwenden Sie Divider sparsam, um die Lesbarkeit zu gewährleisten
- **Konsistenz**: Verwenden Sie konsistente Breiten innerhalb einer Seite
- **Zweck**: Klarer Zweck für jeden Divider (Trennung oder Abstand)
