---
title: Text With Media Section
description: Strukturierter Content-Block für Text-Bild-Kombinationen mit verschiedenen Layout-Optionen
---

Die **Text With Media Section** ist ein strukturierter Content-Block, der Text und Medien (Bilder, Videos, Galerien, Slider) in verschiedenen Layouts kombiniert. Sie eignet sich für Seiten, die strukturierte Text-Bild-Kombinationen benötigen.

## Felder

### Headline

- **Typ**: String
- **Pflichtfeld**: Nein
- **Beschreibung**: Hauptüberschrift für die gesamte Sektion

### Description

- **Typ**: Text
- **Pflichtfeld**: Nein
- **Zeilen**: 3
- **Beschreibung**: Beschreibung für die gesamte Sektion

### Items

- **Typ**: Array von `textWithMediaItem`-Objekten
- **Pflichtfeld**: Ja
- **Minimale Anzahl**: 1
- **Maximale Anzahl**: 10
- **Beschreibung**: Liste von Text-Media-Items

### Background Color

- **Typ**: Reference zu `backgroundColor`
- **Pflichtfeld**: Nein
- **Beschreibung**: Hintergrundfarbe für die gesamte Sektion

## Text With Media Item

Jedes Item in der Sektion kann individuell konfiguriert werden:

### Headline

- **Typ**: String
- **Pflichtfeld**: Nein
- **Beschreibung**: Hauptüberschrift für dieses Item

### Content

- **Typ**: Rich Text Section (`richTextSection`)
- **Pflichtfeld**: Nein
- **Beschreibung**: Textinhalt in Rich-Text-Format

### Layout Options

- **Typ**: Object
- **Pflichtfeld**: Nein
- **Beschreibung**: Layout-Konfiguration für Text und Medien

#### Layout Type

- **Typ**: String
- **Initialwert**: `split`
- **Optionen**:
  - **Split Layout** (`split`) - Text und Medien nebeneinander
  - **Media Above, Text Below** (`mediaAbove`) - Medien oben, Text unten
  - **Text Above, Media Below** (`textAbove`) - Text oben, Medien unten

#### Split Ratio

- **Typ**: String
- **Initialwert**: `50-50`
- **Versteckt**: Ja (wenn Layout Type nicht "Split Layout" ist)
- **Optionen**:
  - **50/50** - Gleichmäßige Aufteilung
  - **60/40** - 60% Text, 40% Medien (oder umgekehrt je nach Position)
- **Beschreibung**: Aufteilungsverhältnis für Split Layout

#### Split Media Position

- **Typ**: String
- **Initialwert**: `left`
- **Versteckt**: Ja (wenn Layout Type nicht "Split Layout" ist)
- **Optionen**:
  - **Media Left, Text Right** (`left`) - Medien links, Text rechts
  - **Media Right, Text Left** (`right`) - Medien rechts, Text links
- **Beschreibung**: Position der Medien relativ zum Text

#### Media Type

- **Typ**: String
- **Initialwert**: `image`
- **Optionen**:
  - **Single Image** (`image`) - Einzelnes Bild
  - **Image Gallery** (`imageGallery`) - Bildergalerie
  - **Video** (`video`) - Video
  - **Image Slider** (`slider`) - Bilderslider
- **Beschreibung**: Art der Medien, die neben dem Text angezeigt werden sollen

### Media Fields

Je nach gewähltem Media Type werden unterschiedliche Felder angezeigt:

#### Single Image

- **Typ**: Image
- **Versteckt**: Ja (wenn Media Type nicht "Single Image" ist)
- **Optionen**: Hotspot aktiviert
- **Felder**:
  - **Alternative Text** (String) - Wichtig für Barrierefreiheit und SEO
  - **Caption** (String) - Optionale Bildunterschrift

#### Image Gallery

- **Typ**: Array von Bildern
- **Versteckt**: Ja (wenn Media Type nicht "Image Gallery" ist)
- **Minimale Anzahl**: 1
- **Maximale Anzahl**: 20
- **Felder pro Bild**:
  - **Alt Text** (String) - Wichtig für Barrierefreiheit und SEO
  - **Caption** (String) - Optionale Bildunterschrift

#### Video

- **Typ**: Object
- **Versteckt**: Ja (wenn Media Type nicht "Video" ist)
- **Felder**:
  - **Video File** (File) - Hochgeladenes Video (MP4, WebM, etc.)
  - **Video URL** (URL) - Alternative: Direkte Video-URL
  - **Poster Image** (Image) - Thumbnail-Bild, das vor dem Abspielen angezeigt wird

#### Image Slider

- **Typ**: Object
- **Versteckt**: Ja (wenn Media Type nicht "Image Slider" ist)
- **Felder**:
  - **Slides** (Array, 1-10 Slides)
    - **Slide Image** (Image, Pflichtfeld)
    - **Title** (String, max. 100 Zeichen)
    - **Description** (Text, max. 300 Zeichen)
    - **Call to Action Buttons** (Reference zu `button`)
  - **Enable Autoplay** (Boolean, Initialwert: `true`)
  - **Autoplay Speed** (Number, 1-30 Sekunden)
  - **Show Navigation Dots** (Boolean, Initialwert: `true`)

### Call to Action Buttons

- **Typ**: Reference zu `button`
- **Pflichtfeld**: Nein
- **Beschreibung**: Button-Konfiguration für dieses Item

## Verwendung

Die Text With Media Section wird typischerweise verwendet für:

- **Produktpräsentationen** - Produkte mit Bildern und Beschreibungen
- **Dienstleistungsseiten** - Dienstleistungen mit visuellen Elementen
- **Über-uns-Seiten** - Informationen über das Unternehmen mit Bildern
- **Feature-Präsentationen** - Features mit erklärenden Bildern oder Videos
- **Angebotsseiten** - Angebote mit visueller Unterstützung

## Beispiel

Eine typische Text With Media Section könnte folgende Struktur haben:

### Headline

"Unsere Fahrzeuge"

### Description

"Moderne und umweltfreundliche Fahrzeuge für Ihre Ausbildung"

### Items

**Item 1:**

- **Headline**: "Elektrofahrzeuge"
- **Content**: "Lernen Sie das Fahren mit modernen Elektrofahrzeugen..."
- **Layout Type**: Split Layout
- **Split Ratio**: 50/50
- **Media Position**: Left
- **Media Type**: Single Image
- **Image**: Bild eines Elektrofahrzeugs

**Item 2:**

- **Headline**: "Fahrzeugflotte"
- **Content**: "Unsere vielfältige Fahrzeugflotte..."
- **Layout Type**: Media Above, Text Below
- **Media Type**: Image Gallery
- **Images**: Array von Fahrzeugbildern

**Item 3:**

- **Headline**: "Fahrstunden-Video"
- **Content**: "Sehen Sie, wie eine typische Fahrstunde abläuft..."
- **Layout Type**: Text Above, Media Below
- **Media Type**: Video
- **Video**: Video-Datei oder URL

## Layout-Optionen im Detail

### Split Layout

- **50/50 Split**: Gleichmäßige Aufteilung zwischen Text und Medien
- **60/40 Split**: Größerer Bereich für Text oder Medien
- **Media Position**: Links oder rechts positionierbar

### Media Above, Text Below

Medien werden vollständig über dem Text angezeigt. Ideal für:

- Große, aussagekräftige Bilder
- Videos, die Aufmerksamkeit erregen sollen
- Galerien, die prominent präsentiert werden sollen

### Text Above, Media Below

Text wird über den Medien angezeigt. Ideal für:

- Textlastige Inhalte
- Erklärungen vor visuellen Elementen
- Schritt-für-Schritt-Anleitungen

## Besonderheiten

- **Flexible Layouts**: Drei verschiedene Layout-Typen für unterschiedliche Anforderungen
- **Verschiedene Media-Typen**: Einzelbilder, Galerien, Videos oder Slider
- **Rich Text Content**: Vollständiger Rich-Text-Editor für Textinhalte
- **Mehrere Items**: Bis zu 10 Items pro Sektion für umfangreiche Inhalte
- **Individuelle Konfiguration**: Jedes Item kann individuell konfiguriert werden

## Best Practices

- **Konsistenz**: Verwenden Sie ähnliche Layout-Typen innerhalb einer Sektion für bessere Lesbarkeit
- **Media-Auswahl**: Wählen Sie den passenden Media-Typ für den Inhalt
- **Alt-Text**: Immer Alt-Text für Bilder ausfüllen für Barrierefreiheit und SEO
- **Split Ratio**: 50/50 für ausgewogene Darstellung, 60/40 für text- oder bildlastige Inhalte
- **Anzahl der Items**: 3-5 Items sind optimal für die Benutzererfahrung
- **Hintergrundfarbe**: Verwenden Sie Hintergrundfarben sparsam, um die Lesbarkeit zu gewährleisten

## Unterschiede zu anderen Content-Blöcken

| Feature         | Text With Media Section     | Rich Text Section | Gallery Section |
| --------------- | --------------------------- | ----------------- | --------------- |
| Struktur        | Fest (Text + Media)         | Flexibel          | Nur Medien      |
| Layout-Optionen | Ja (3 Typen)                | Nein              | Nein            |
| Media-Typen     | Ja (4 Typen)                | Nur Bilder        | Nur Bilder      |
| Rich Text       | Ja (pro Item)               | Ja (vollständig)  | Nein            |
| Mehrere Items   | Ja (1-10)                   | Nein              | Nein            |
| Videos          | Ja                          | Nein              | Nein            |
| Slider          | Ja                          | Nein              | Nein            |
| Verwendung      | Strukturierte Kombinationen | Fließtext         | Bildergalerien  |
