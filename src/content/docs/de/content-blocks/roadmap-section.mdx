title: Roadmap Section
description: Content-Block zur Darstellung von Roadmaps und Timelines
---

Die **Roadmap Section** ist ein Content-Block zur Darstellung von Roadmaps, Timelines oder Meilensteinen in chronologischer Reihenfolge.

## Felder

### Headline

- **Typ**: String
- **Pflichtfeld**: Ja
- **Maximale Länge**: 100 Zeichen
- **Beschreibung**: Hauptüberschrift für die Roadmap-Sektion

### Description

- **Typ**: Text
- **Pflichtfeld**: Nein
- **Zeilen**: 3
- **Maximale Länge**: 500 Zeichen
- **Beschreibung**: Beschreibung für die Roadmap-Sektion

### Timeline Items

- **Typ**: Array von Timeline-Item-Objekten
- **Pflichtfeld**: Nein
- **Minimale Anzahl**: 1
- **Maximale Anzahl**: 20
- **Beschreibung**: Timeline-Items für die Roadmap

#### Timeline-Item Felder

##### Title

- **Typ**: String
- **Pflichtfeld**: Ja
- **Maximale Länge**: 100 Zeichen
- **Beschreibung**: Titel des Meilensteins oder Features

##### Description

- **Typ**: Text
- **Pflichtfeld**: Nein
- **Zeilen**: 3
- **Maximale Länge**: 500 Zeichen
- **Beschreibung**: Detaillierte Beschreibung des Meilensteins

### Background Color

- **Typ**: Reference zu `backgroundColor`
- **Pflichtfeld**: Nein
- **Beschreibung**: Hintergrundfarbe für die gesamte Roadmap-Sektion

## Verwendung

Die Roadmap Section wird typischerweise verwendet für:

- **Projekt-Roadmaps** - Darstellung von Projektplänen
- **Timelines** - Chronologische Darstellung von Ereignissen
- **Meilensteine** - Wichtige Meilensteine und Ziele
- **Entwicklungspläne** - Pläne für zukünftige Entwicklungen

## Beispiel

Eine typische Roadmap Section könnte folgende Struktur haben:

### Headline

"Unsere Roadmap"

### Description

"Unsere geplanten Meilensteine und Entwicklungen"

### Timeline Items

**Item 1:**

- **Title**: "Q1 2025: Neue Fahrzeuge"
- **Description**: "Integration von Elektrofahrzeugen in unsere Flotte"

**Item 2:**

- **Title**: "Q2 2025: Online-Anmeldung"
- **Description**: "Einführung einer Online-Anmeldefunktion"

**Item 3:**

- **Title**: "Q3 2025: Erweiterte Standorte"
- **Description**: "Eröffnung neuer Standorte in der Region"

## Best Practices

- **Chronologie**: Items in chronologischer Reihenfolge anordnen
- **Klarheit**: Klare, prägnante Titel
- **Beschreibungen**: Detaillierte, aber prägnante Beschreibungen
- **Anzahl**: 3-10 Items sind optimal
- **Aktualität**: Roadmap regelmäßig aktualisieren
