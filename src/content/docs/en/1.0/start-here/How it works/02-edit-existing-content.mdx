Learn how to find, edit, and publish existing website content in Sanity Studio.

<br />

<Steps>
  1. **Navigate to the content you want to edit.**
     For example, in the left sidebar, click **Pages** to access all website pages.

  2. **Select the content type.**
     Under **Pages**, click **Page** to see the list of available pages.

  3. **Choose the specific page you want to edit.**
     The page editor will open on the right side of the screen.

  4. **Select the component you want to modify.**
     Scroll through the page content and click on the component you'd like to change.

     <figure style="text-align: center;">
       <img src={sanityStudioWorkflowExample.src} alt="Sanity Studio edit existing content example" />

       <figcaption>
         <em>Sanity Studio edit existing content example</em>
       </figcaption>
     </figure>

     <br />

  5. **Edit the component content.**
     The component editor will open. Click into any text field, image, or setting to make your changes. Your edits are saved automatically as a draft.

     <figure style="text-align: center;">
       <img src={sanityStudioEditComponent.src} alt="Sanity Studio edit component content example" />

       <figcaption>
         <em>Sanity Studio edit component content example</em>
       </figcaption>
     </figure>

     <br />

  6. **Edit items within a component (if applicable).**
     Some components contain multiple items (like team members in a Team section, or images in a Gallery). Click on an individual item to edit it.

     <figure style="text-align: center;">
       <img src={sanityStudioEditComponentItem.src} alt="Sanity Studio edit existing component item example" />

       <figcaption>
         <em>Sanity Studio edit existing component item example</em>
       </figcaption>
     </figure>

     <br />

  7. **Publish your changes.**
     Once you're satisfied with your edits, click the **Publish** button at the bottom of the editor to make your changes visible on the live website.
</Steps>
