The Map Section provides an interactive map with configurable coordinates, zoom level, and markers. Perfect for showing your business location, helping visitors find you, and supporting route planning.

<Aside>
  **Best for:** Location display on contact pages, multiple locations, route
  planning support, business finders, location-based services.
</Aside>

## Fields

### Latitude

* **Type:** Number (required)
* **Min Value:** -90
* **Max Value:** 90
* **Default Value:** 48.553752
* **Purpose:** Latitude coordinate for the map center point

### Longitude

* **Type:** Number (required)
* **Min Value:** -180
* **Max Value:** 180
* **Default Value:** 11.271375
* **Purpose:** Longitude coordinate for the map center point

### Zoom Level

* **Type:** Number (required)
* **Min Value:** 1
* **Max Value:** 20
* **Default Value:** 17
* **Purpose:** Initial zoom level for the map (1-20)

### Marker Title

* **Type:** String (required)
* **Default Value:** "Example Company"
* **Purpose:** Title displayed in the marker popup
* **Example:** "Example Company"

### Marker Address

* **Type:** String (required)
* **Default Value:** "Example Street 15"
* **Purpose:** Address displayed in the marker popup
* **Example:** "123 Main Street"

### Marker City

* **Type:** String (required)
* **Default Value:** "12345 Example City"
* **Purpose:** City displayed in the marker popup
* **Example:** "12345 Example City"

### Enable Scroll Wheel Zoom

* **Type:** Boolean (optional)
* **Default Value:** `true`
* **Purpose:** Allows users to zoom with the mouse wheel

***

## Usage Example

Here's a typical Map Section configuration:

* **Latitude:** 48.553752
* **Longitude:** 11.271375
* **Zoom Level:** 17
* **Marker Title:** "Example Company"
* **Marker Address:** "123 Main Street"
* **Marker City:** "12345 Example City"
* **Enable Scroll Wheel Zoom:** `true`

## Best Practices

**Coordinates:** Use accurate GPS coordinates for your location—you can find these using Google Maps or similar services

**Zoom level:** 15-18 for city view, 17-20 for detailed view

**Marker information:** Provide complete address information in the marker

**Scroll zoom:** Enable for better user experience and accessibility

**Testing:** Test the map on different devices to ensure it displays correctly

**Accessibility:** Ensure marker information is clear and complete

***

## Key Features

**Interactive map:** Users can zoom, pan, and interact with the map

**Customizable marker:** Title, address, and city displayed in marker popup

**Zoom control:** Configurable initial zoom level

**Scroll wheel zoom:** Optional mouse wheel zooming for better navigation

**Responsive:** Map adapts to different screen sizes

***

## Where It's Used

The Map Section appears in these content types:

* **Pages** (`page`) – Contact pages and location pages
* **Contact Information Section** – Referenced within contact information sections

***

## Technical Notes

* Coordinates use decimal degrees format (WGS84)
* Zoom level 1 shows the entire world, 20 shows street-level detail
* Marker popup displays when users click on the marker
* Scroll wheel zoom can be disabled for specific use cases
* The map uses standard mapping APIs for rendering
