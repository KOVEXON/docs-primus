The Masonry Gallery Section displays images in a flexible, staggered masonry layout—perfect for creating visually interesting galleries where images have different aspect ratios. Unlike traditional grids, masonry layouts adapt to image sizes, creating a dynamic, Pinterest-style presentation.

<Aside>
  **Best for:** Portfolio galleries, event galleries, product galleries with
  varying image sizes, creative presentations, inspiration galleries.
</Aside>

## Fields

### Gallery Images

* **Type:** Array of gallery image objects (required)
* **Min Count:** 1
* **Max Count:** 24
* **Purpose:** List of gallery images

#### Gallery Image Fields

* **Image** (Image, required) – The image with hotspot support
* **Alt Text** (String) – Alternative text for the image (overrides image alt text)
* **Caption** (String) – Optional image caption displayed on hover

### Number of Columns

* **Type:** Object (optional)
* **Purpose:** Column configuration for different screen sizes

#### Mobile Columns

* **Type:** Number (optional)
* **Default Value:** 2
* **Min Value:** 1
* **Max Value:** 3
* **Purpose:** Number of columns on mobile devices

#### Desktop Columns

* **Type:** Number (optional)
* **Default Value:** 4
* **Min Value:** 2
* **Max Value:** 6
* **Purpose:** Number of columns on desktop devices

### Images Per Column

* **Type:** Number (optional)
* **Default Value:** 3
* **Min Value:** 1
* **Max Value:** 10
* **Purpose:** Maximum number of images per column

### Enable Image Modal

* **Type:** Boolean (optional)
* **Default Value:** `true`
* **Purpose:** Allows users to view images in a modal by clicking

***

## Usage Example

Here's a typical Masonry Gallery Section configuration:

* **Mobile Columns:** 2
* **Desktop Columns:** 4
* **Images Per Column:** 3
* **Enable Image Modal:** `true`
* **Images:** Array of 12-20 images with varying sizes

***

## Best Practices

**Image sizes:** Use images with varying aspect ratios for a more interesting layout

**Alt text:** Always fill in meaningful alt text for each image for accessibility and SEO

**Count:** 12-20 images are optimal for the masonry layout

**Columns:** 2-3 columns on mobile, 4-6 on desktop are good guidelines

**Captions:** Use captions to provide context when images are hovered

**Consistency:** Maintain consistent image quality even with varying sizes

***

## Key Features

**Masonry layout:** Flexible, staggered arrangement of images

**Responsive:** Different column counts for mobile and desktop

**Modal view:** Images can be viewed in an enlarged modal

**Hover captions:** Captions displayed when hovering over images

**Flexible sizing:** Adapts to images of different sizes and aspect ratios

***

## Where It's Used

The Masonry Gallery Section appears in these content types:

* **Pages** (`page`) – General content pages, especially portfolio and event pages

***

## Technical Notes

* Masonry layout automatically adjusts to image sizes
* Images are loaded lazily for better performance
* Modal supports keyboard navigation for accessibility
* Captions are only visible on hover/click
* Background color is not configurable for this section
* Images support hotspot editing for better cropping
