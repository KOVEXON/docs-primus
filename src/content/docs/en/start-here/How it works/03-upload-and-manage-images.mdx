Learn how to upload, replace, and manage images in Sanity Studio, including adding alt text for accessibility and SEO.

<br />

<Steps>

1. **Click on the image field** you want to edit (usually shows a placeholder or existing image).

2. **Upload or select an image:**

   - Click **Upload** to add a new image from your computer
   - Or click **Select** to choose an existing image from the media library

3. **Add alt text (recommended).**
   If an alt text field is available, add a brief description of the image. This helps people using screen readers and improves search engine visibility (SEO).

   {" "}

   <Aside type="tip">
     Good alt text example: "Team meeting in bright office space" rather than
     just "office"
   </Aside>

4. **Publish your changes** to make the new image visible on the website.

</Steps>
