The Social Media Links Section displays social media links with platform-specific icons. With support for numerous platforms, custom icons, flexible alignment, and size options, it's perfect for connecting visitors to your social media presence.

<Aside>
  **Best for:** Footer social links, contact pages, about-us pages, dedicated
  social media areas, brand presence sections.
</Aside>

## Fields

### Headline

- **Type:** String (optional)
- **Max Length:** 100 characters
- **Purpose:** Optional heading for the social links section
- **Example:** "Follow Us" or "Connect With Us"

### Description

- **Type:** Text, 3 rows (optional)
- **Max Length:** 500 characters
- **Purpose:** Optional description
- **Example:** "Stay up to date"

### Social Links

- **Type:** Array of social link objects (optional)
- **Min Count:** 1
- **Max Count:** 20
- **Purpose:** List of social media links

#### Social Link Fields

**Platform** (required)

- Social media platform
- Default: `facebook`
- **Options:**
  - Facebook
  - Twitter/X
  - Instagram
  - LinkedIn
  - YouTube
  - TikTok
  - Pinterest
  - Snapchat
  - WhatsApp
  - Telegram
  - GitHub
  - Custom

**URL** (required)

- Complete URL to the social media profile or page
- Example: `https://facebook.com/example-company`

**Label** (optional)

- Optional custom label (for screen readers and tooltips)
- Example: "Follow us on Facebook"

**Custom Icon** (optional, hidden when Platform is not "Custom")

- Optional: Name of Lucide icon from https://lucide.dev/icons
- Only used when Platform is "Custom"

### Alignment

- **Type:** String (optional)
- **Default Value:** `center`
- **Options:**
  - **Left** (`left`)
  - **Center** (`center`)
  - **Right** (`right`)
- **Purpose:** Horizontal alignment of social links

### Icon Size

- **Type:** String (optional)
- **Default Value:** `medium`
- **Options:**
  - **Small** (`small`)
  - **Medium** (`medium`)
  - **Large** (`large`)
- **Purpose:** Size of social media icons

---

## Usage Example

Here's a typical Social Media Links Section configuration:

- **Headline:** "Follow Us"
- **Description:** "Stay up to date"
- **Social Links:**
  - Facebook: `https://facebook.com/example-company`
  - Instagram: `https://instagram.com/example_company`
  - YouTube: `https://youtube.com/@example-company`
- **Alignment:** Center
- **Icon Size:** Medium

---

## Best Practices

**Relevance:** Only link to active social media profiles

**Consistency:** Use consistent icon sizes throughout

**Alignment:** Center alignment works well for most layouts

**Count:** 3-6 links are optimal for user experience

**Labels:** Use labels for better accessibility and clarity

**Platform selection:** Choose platforms that match your audience and content strategy

---

## Key Features

**Multiple platforms:** Support for numerous social media platforms

**Custom icons:** Option for custom icons when needed

**Flexible alignment:** Left, center, or right alignment options

**Size options:** Small, medium, or large icon sizes

**Accessibility:** Labels support screen readers and tooltips

---

## Where It's Used

The Social Media Links Section appears in these content types:

- **Pages** (`page`) – General content pages, especially footer areas and contact pages

---

## Technical Notes

- Icons are automatically selected based on platform
- Custom icons use Lucide icon library when platform is set to "Custom"
- Links open in the same tab by default (can be configured)
- Icon sizes are responsive and adapt to screen size
- Background color is not configurable for this section
