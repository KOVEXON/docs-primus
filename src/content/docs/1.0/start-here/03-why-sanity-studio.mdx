Ihre Website wurde mit Sanity Studio erstellt, damit Sie:

* Inhalte **selbstständig** aktualisieren können
* Technische Komplexität vermeiden
* Die Website **konsistent und fehlerfrei** halten
* Zeit und Kosten für kleine Änderungen sparen
